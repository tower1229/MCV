# V0.1 Canonical content limited to three items

The `common/` directory in v0.1 contains only three items: `AGENTS.md` (unified rules), `skills/` (SKILL.md directory packages), and `mcp.yaml` (MCP server registry). `workflows/` and `preferences.yaml` are deferred — their formats are undefined, and shipping empty placeholders adds no value. They can be introduced in later versions when concrete requirements emerge.
